## ALX Listing App

The goal of this project is to scaffold the Airbnb clone app using Next.js with TypeScript integration following the [Figma file](https://www.figma.com/design/E2BRqdPcKkrnX6hLGPto8Z/Project-Airbnb?node-id=1-4&t=nrHtrJrOvJpF16IY-1).

## API Setup

| Endpoint | HTTP Method | Description |
| --- | --- | --- |
| /properties | GET | Fetches the list of properties for the listing page |
| /properties/:id | GET | Fetches the details of a single property based on its ID |
| /bookings | POST | Submits booking details to the server |
| /properties/:id/reviews | GET | Fetches the reviews of a property |
